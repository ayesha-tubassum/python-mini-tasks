# Python Mini Projects 

This repository contains my final Python project from SMIT AI & Data Science course.

## Project Overview
This project includes multiple problem-solving tasks and mini systems built using Python.

---

## Features

### 1. Number Analysis Program
- Find minimum and maximum (without built-in functions)
- Calculate average
- Count even and odd numbers
- Find numbers greater than average
- Reverse list manually

### 2. Library Management System
- Search books
- Add books
- Remove books
- Display all books
- User-based book list system

### 3. Recursion Problems
- Factorial calculation
- Fibonacci sequence
- Sum of list (recursive)
- Palindrome checker
- Power function
